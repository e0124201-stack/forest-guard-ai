# Forest Guard AI
### Real-time Illegal Logging and Fire Alert System

This Machine Learning project uses Computer Vision and Audio Processing to protect forests from wildfires and illegal activities.

## Features
- Fire Detection: Monitors live camera feeds for fire patterns using OpenCV.
- Logging Detection: Analyzes audio frequencies to detect chainsaw sounds using Librosa.
- Alert System: Provides real-time status updates and emergency logging.

## Tech Stack
- Python
- Streamlit (Dashboard UI)
- OpenCV (Image Processing)
- Librosa (Audio Analysis)
- Matplotlib (Data Visualization)
- NumPy (Data Processing)
